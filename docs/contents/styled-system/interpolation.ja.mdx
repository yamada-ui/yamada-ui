$(インターポレーション)は、スタイルシステム内で[CSSカスタムプロパティ(変数)](/styled-system/css-custom-properties)、または[テーマ](/styled-system/theming)のトークンを参照するための機能です。

## 使い方

[Style Props](/styled-system/style-props)の`vars`を使用して、[CSSカスタムプロパティ(変数)](/styled-system/css-custom-properties)を作成した場合、設定した`name`を`$`で参照することができます。

```tsx
<Box
  p="md"
  rounded="md"
  color="white"
  vars={[{ name: "bg", token: "colors", value: "primary" }]}
  bg="$bg"
>
  Box
</Box>
```

## テーマのトークンを参照する

Yamada UIの[Style Props](/styled-system/style-props)は、対応するプロパティのみ[テーマ](/styled-system/theming)のトークンが参照されます。例えば、`borderRadius`は、`radii`のトークンを参照しますが、`spaces`のトークンは参照されません。`$`を使用することで、対応するトークン以外を参照することができます。

```tsx
<Box p="md" rounded="$spaces.3" bg="primary" color="white">
  Box
</Box>
```

また、文字列内に`$`を使用することで、[テーマ](/styled-system/theming)のトークンを参照することができます。これは、[border](https://developer.mozilla.org/ja/docs/Web/CSS/border)などの[一括指定プロパティ](https://developer.mozilla.org/ja/docs/Web/CSS/Shorthand_properties)で有効です。

```tsx
<Box p="md" rounded="md" border="1px solid $colors.primary" color="primary">
  Box
</Box>
```
