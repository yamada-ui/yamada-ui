`FormControl`を使用しない場合は、`NumberInput`に`aria-label`または`aria-labelledby`を設定します。

```tsx live=false
<NumberInput aria-label="Age" />
```

```tsx live=false
<VStack gap="sm">
  <Text as="h3" id="label">
    Age
  </Text>

  <NumberInput aria-labelledby="label" />
</VStack>
```

例えば、曜日が数値で表されており、`aria-valuenow`の値がユーザーフレンドリーではない場合は、`aria-valuetext`に`"Monday"`などの人間が理解可能な文字列を設定します。

```tsx live=false
const weekday = [
  "Sunday",
  "Monday",
  "Tuesday",
  "Wednesday",
  "Thursday",
  "Friday",
  "Saturday",
]

return (
  <NumberInput
    getAriaValueText={(value) => weekday[Number(value)]}
    max={6}
    min={0}
  />
)
```

## キーボード操作

| キー        | 説明                                 | 状態 |
| ----------- | ------------------------------------ | ---- |
| `ArrowUp`   | `step`の値に基づいて値を増加します。 | -    |
| `ArrowDown` | `step`の値に基づいて値を減少します。 | -    |
| `Home`      | `min`の値を設定します。              | -    |
| `End`       | `max`の値を設定します。              | -    |

## ARIAロールと属性

| コンポーネント                           | ロールと属性        | 使い方                                                             |
| ---------------------------------------- | ------------------- | ------------------------------------------------------------------ |
| `NumberInput`                            | `role="group"`      | グループであることを示します。                                     |
| `NumberInputField` <Tag>内部</Tag>       | `role="spinbutton"` | スピンボタンであることを示します。                                 |
|                                          | `aria-valuemin`     | `min`の値を設定します。デフォルトは`Number.MIN_SAFE_INTEGER`です。 |
|                                          | `aria-valuemax`     | `max`の値を設定します。デフォルトは`Number.MAX_SAFE_INTEGER`です。 |
|                                          | `aria-valuenow`     | 現在の値を設定します。                                             |
|                                          | `aria-valuetext`    | 現在の値を設定します。                                             |
| `NumberIncrementStepper` <Tag>内部</Tag> | `aria-label`        | `"Increase"`を設定します。                                         |
| `NumberDecrementStepper` <Tag>内部</Tag> | `aria-label`        | `"Decrease"`を設定します。                                         |
