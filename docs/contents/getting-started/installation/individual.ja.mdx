## パッケージをインストールする

Yamada UIのコアやユーティリティをインストールします。

ターミナルで次のコマンドのいずれかを実行します。

<PackageManagers
  packageNameOrCommand={[
    "@yamada-ui/core",
    "@yamada-ui/theme",
    "@yamada-ui/utils",
  ]}
/>

次に、プロジェクトに使用したいコンポーネントやフックをインストールしていきます。

ここでは、`Box`などを含んでいる`@yamada-ui/layouts`をインストールしてみましょう。

<PackageManagers packageNameOrCommand="@yamada-ui/layouts" />

:::note
コンポーネントやフックのパッケージ名は、各ドキュメントページに記載されています。
:::

## `ThemeProvider`を追加する

コアやコンポーネントをインストール後、アプリケーションのルートに`ThemeProvider`, `ResetStyle`, `GlobalStyle`を追加し、`@yamada-ui/theme`から[デフォルトのテーマ](/styled-system/theming/default-theme)と[デフォルトのコンフィグ](/styled-system/configure/default-config)を渡します。もし、あなたのテーマやコンフィグがある場合は、そちらを渡します。

```tsx live=false highlight={6,7,8,11}
import { ThemeProvider, ResetStyle, GlobalStyle } from "@yamada-ui/core"
import { defaultTheme, defaultConfig } from "@yamada-ui/theme"

const App = () => {
  return (
    <ThemeProvider theme={defaultTheme} config={defaultConfig}>
      <ResetStyle />
      <GlobalStyle />

      <YourApplication />
    </ThemeProvider>
  )
}
```

ダークモードを対応させたい場合は、`ColorModeProvider`も追加します。

```tsx live=false highlight={7,12}
import {
  ThemeProvider,
  ColorModeProvider,
  ResetStyle,
  GlobalStyle,
} from "@yamada-ui/core"
import { defaultTheme, defaultConfig } from "@yamada-ui/theme"

const App = () => {
  return (
    <ThemeProvider theme={defaultTheme} config={defaultConfig}>
      <ColorModeProvider config={defaultConfig}>
        <ResetStyle />
        <GlobalStyle />

        <YourApplication />
      </ColorModeProvider>
    </ThemeProvider>
  )
}
```

:::note
テーマやコンフィグをカスタマイズしたい場合は、[テーマをカスタマイズする](/styled-system/theming/customize-theme)と[コンフィグをカスタマイズする](/styled-system/configure/customize-config)をご覧ください。
:::

:::note status=warning
`theme`や`config`を`ThemeProvider`に渡さない場合、コンポーネントのスタイルやトークンは反映されません。
:::

## コンポーネントを使用する

アプリケーション内でコンポーネントを呼び出します。

```tsx live=false
import { Box } from "@yamada-ui/layouts"

const Demo = () => {
  return <Box>This is Box</Box>
}
```

これで、Yamada UIのセットアップは完了です！
