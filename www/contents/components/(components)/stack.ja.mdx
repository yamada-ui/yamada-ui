## 使い方

:::code-group

```tsx [package]
import { Stack } from "@yamada-ui/react"
```

```tsx [alias]
import { Stack } from "@/components/ui"
```

```tsx [monorepo]
import { Stack } from "@workspaces/ui"
```

:::

```tsx
<Stack />
```

::::tip

::::

::::tip
横方向に積み重ねたい場合は、[HStack](/docs/components/h-stack)の使用を検討してください。
::::

::::tip
要素を重ねて配置したい場合は、[ZStack](/docs/components/z-stack)の使用を検討してください。
::::

### 子要素間に区切り線を設ける

```tsx preview
<Stack separator={<Separator />}>
  {Array.from({ length: 4 }).map((_, index) => (
    <Box key={index} bg="bg.contrast" color="fg.contrast" p="md">
      Box
    </Box>
  ))}
</Stack>
```
