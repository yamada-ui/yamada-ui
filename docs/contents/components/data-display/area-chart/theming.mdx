## Theming

The `AreaChart` is a [multi part component](/styled-system/theming/component-styles#multi-part-components).

:::note
`AreaChart` inherits the style from [LineChart](/components/data-display/line-chart).
:::

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::

```ts
export const AreaChart: ComponentMultiStyle<"AreaChart"> = mergeMultiStyle(
  LineChart,
  {
    baseStyle: {
      area: {},
    },
  },
)({ omit: ["line"] })
```
