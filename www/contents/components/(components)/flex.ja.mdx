```tsx preview
<Flex gap="md">
  {Array.from({ length: 4 }).map((_, index) => (
    <Box key={index} bg="bg.contrast" color="fg.contrast" p="md">
      Box
    </Box>
  ))}
</Flex>
```

## 使い方

:::code-group

```tsx [package]
import { Flex } from "@yamada-ui/react"
```

```tsx [alias]
import { Flex } from "@/components/ui"
```

```tsx [monorepo]
import { Flex } from "@workspaces/ui"
```

:::

```tsx
<Flex />
```

::::tip
汎用的なフレックスレイアウトではなく、縦方向または横方向の積み重ねを既定のスタイル付きで使いたい場合は、[VStack](/docs/components/v-stack)または[HStack](/docs/components/h-stack)の使用を検討してください。`VStack`はデフォルトで縦方向に並び、子要素が伸長されて幅いっぱいに表示されます。`HStack`はデフォルトで横方向に並び、交差軸方向で中央揃えになります。
::::

::::tip
子要素を複数行に折り返して配置したい場合は、[Wrap](/docs/components/wrap)の使用を検討してください。
::::

### 方向

```tsx preview
<Flex direction="column" gap="md">
  {Array.from({ length: 4 }).map((_, index) => (
    <Box key={index} bg="bg.contrast" color="fg.contrast" p="md">
      Box
    </Box>
  ))}
</Flex>
```

### 整列

```tsx preview
<Flex align="center" gap="md">
  <For each={["sm", "md", "lg"]}>
    {(p, index) => (
      <Box key={index} bg="bg.contrast" color="fg.contrast" p={p}>
        Box
      </Box>
    )}
  </For>
</Flex>
```

### 揃え方

```tsx preview
<Flex justify="space-between" gap="md">
  {Array.from({ length: 4 }).map((_, index) => (
    <Box key={index} bg="bg.contrast" color="fg.contrast" p="md">
      Box
    </Box>
  ))}
</Flex>
```

### 折り返し

```tsx preview
<Flex wrap="wrap" gap="md" maxW="xs">
  {Array.from({ length: 4 }).map((_, index) => (
    <Box key={index} bg="bg.contrast" color="fg.contrast" p="md">
      Box
    </Box>
  ))}
</Flex>
```

### スペースを追加する

```tsx preview
<Flex w="full" gap="md">
  <Box bg="bg.contrast" color="fg.contrast" p="md">
    Box
  </Box>

  <Spacer />

  <Box bg="bg.contrast" color="fg.contrast" p="md">
    Box
  </Box>
</Flex>
```
