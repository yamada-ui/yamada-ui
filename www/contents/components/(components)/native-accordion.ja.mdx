## 使い方

:::note
`NativeAccordion`は、HTMLの`details`と`summary`の使用しており、[Accordion](/components/accordion)と比較して、パフォーマンスとアクセシビリティが向上しています。
:::

:::code-group

```tsx [package]
import { NativeAccordion } from "@yamada-ui/react"
```

```tsx [alias]
import { NativeAccordion } from "@/components/ui"
```

```tsx [monorepo]
import { NativeAccordion } from "@workspaces/ui"
```

:::

```tsx
<NativeAccordion.Root>
  <NativeAccordion.Item>
    <NativeAccordion.Button />
    <NativeAccordion.Panel />
  </NativeAccordion.Item>
</NativeAccordion.Root>
```

## アクセシビリティ

`NativeAccordion`は、アクセシビリティに関して[WAI-ARIA - Disclosure Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/disclosure/)に従います。ネイティブHTMLの`<details>`と`<summary>`要素を使用しており、ブラウザの組み込みのディスクロージャー動作を提供します。

### キーボード操作

| キー             | 説明                                                                                                                         | 状態 |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------- | ---- |
| `Tab`            | フォーカスがアコーディオンに移動すると最初の項目をフォーカスします。アコーディオン内である場合は次の項目をフォーカスします。 | -    |
| `Space`, `Enter` | フォーカスしている項目のパネルを展開し、展開されている場合は折りたたみます。                                                 | -    |

### ARIAロールと属性

| コンポーネント           | ロールと属性           | 使い方                                           |
| ------------------------ | ---------------------- | ------------------------------------------------ |
| `NativeAccordion.Button` | `aria-disabled="true"` | 項目のパネルが無効な場合は`"true"`を設定します。 |
