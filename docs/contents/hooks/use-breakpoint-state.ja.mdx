## インポート

```ts
import { useBreakpointState } from "@yamada-ui/react"
```

## 使い方

```tsx functional=true
const [value, setValue] = useBreakpointState({
  base: 1,
  md: 2,
  sm: 3,
})

return (
  <Button onClick={() => setValue((prev) => prev + 1)}>
    The current value is "{value}"
  </Button>
)
```

:::note status=warning
オブジェクトのキーは、テーマの[ブレイクポイント](/styled-system/theming/default-theme#breakpoints)で設定されているキーを定義します。
:::

:::note
レスポンシブスタイルをもっと学びたい場合は、[こちら](/styled-system/responsive-styles)をご覧ください。
:::
