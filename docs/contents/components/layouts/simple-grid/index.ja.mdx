## インポート

```ts
import { SimpleGrid, GridItem } from "@yamada-ui/react"
```

## 使い方

:::note
`SimpleGrid`は、内部的に[Grid](/components/layouts/grid)を使用しており、`Grid`を簡素化したコンポーネントです。
:::

```tsx
<SimpleGrid w="full" columns={{ base: 2, md: 1 }} gap="md">
  <For each={["primary", "secondary", "warning", "danger"]}>
    {(bg, index) => (
      <GridItem key={index} w="full" h="4xs" rounded="md" bg={bg} />
    )}
  </For>
</SimpleGrid>
```

### 子要素の最小幅を設定する

子要素の最小幅を設定する場合は、`minChildWidth`に横幅を設定します。

例えば、`120px`を指定した場合、CSSプロパティの`gridTemplateColumns`は、`repeat(auto-fit, minmax(120px, 1fr))`になります。

```tsx
<SimpleGrid w="full" minChildWidth={{ base: "3xs", lg: 100 }} gap="md">
  <For each={["primary", "secondary", "warning", "danger"]}>
    {(bg, index) => (
      <GridItem key={index} w="full" h="4xs" rounded="md" bg={bg} />
    )}
  </For>
</SimpleGrid>
```
