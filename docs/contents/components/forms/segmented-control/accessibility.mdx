## Keyboard Navigation

| Key          | Description                         | State |
| ------------ | ----------------------------------- | ----- |
| `Tab`        | Focuses the selected button.        | -     |
| `ArrowLeft`  | Selects the previous enable button. | -     |
| `ArrowRight` | Selects the next enable button.     | -     |
