## テーマ

`ContextMenu`は、[複数パーツのコンポーネント](/styled-system/theming/component-styles#複数パーツのコンポーネント)です。

:::note
`ContextMenu`は、[Menu](/components/overlay/menu)のスタイルを継承しています。
:::

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const ContextMenu: ComponentMultiStyle = mergeMultiStyle(Menu, {
  baseStyle: {
    trigger: {},
  },
})()
```
