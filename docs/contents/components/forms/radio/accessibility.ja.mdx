Radioコンポーネントは、アクセシビリティに関して[WAI-ARIA - Radio Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/radio)に従います。

## キーボード操作

| キー         | 説明                                                                                                                                                                                  | 状態 |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---- |
| `Tab`        | フォーカスがラジオグループに移動するとチェックされたラジオボタンにフォーカスが移動します。 <br/> ラジオボタンがチェックされていない場合、先頭のラジオボタンにフォーカスを移動します。 | -    |
| `Space`      | フォーカスされたチェックされていないラジオボタンをチェックします。                                                                                                                    | -    |
| `ArrowLeft`  | 無効化されていない前のラジオボタンをチェックします。                                                                                                                                  | -    |
| `ArrowRight` | 無効化されていない次のラジオボタンをチェックします。                                                                                                                                  | -    |
| `ArrowUp`    | 無効化されていない前のラジオボタンをチェックします。                                                                                                                                  | -    |
| `ArrowDown`  | 無効化されていない次のラジオボタンをチェックします。                                                                                                                                  | -    |

## ARIAロールと属性

| コンポーネント | ロールと属性        | 使い方                                                                                                    |
| -------------- | ------------------- | --------------------------------------------------------------------------------------------------------- |
| `RadioGroup`   | `role="radiogroup"` | ラジオボタンのグループであることを示します。                                                              |
|                | `id`                | `Radio`と関連付けるために使用する`id`。                                                                   |
|                | `aria-labelledby`   | `Radio`に関連した`id`を設定します。                                                                       |
| `Radio`        | `role="radio"`      | ラジオボタンであることを示します。                                                                        |
|                | `aria-checked`      | ラジオボタンがチェックされている場合、`true`が設定され、チェックされていない場合は`false`が設定されます。 |
