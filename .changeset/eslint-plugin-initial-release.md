Initial release of `@yamada-ui/eslint-plugin`. Adds the `no-hardcoded-token` rule that reports hard-coded style values (colors, spacing, font sizes, etc.) on Yamada UI components and auto-fixes them to the equivalent design token when one exists. Configurable via the `ignoreScales` and `sources` options.
