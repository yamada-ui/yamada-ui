## Usage

:::note
`NativeAccordion` is a native UI component powered by browser `details`/`summary` behavior. If you need the richer controlled state model, use [Accordion](/components/accordion).
:::

:::code-group

```tsx [package]
import { NativeAccordion } from "@yamada-ui/react"
```

```tsx [alias]
import { NativeAccordion } from "@/components/ui"
```

```tsx [monorepo]
import { NativeAccordion } from "@workspaces/ui"
```

:::

```tsx
<NativeAccordion.Root>
  <NativeAccordion.Item>
    <NativeAccordion.Button />
    <NativeAccordion.Panel />
  </NativeAccordion.Item>
</NativeAccordion.Root>
```

### Use items

```tsx preview functional
const items = useMemo<NativeAccordion.Item[]>(
  () => [
    {
      button: "孫悟空少年編",
      children:
        "地球の人里離れた山奥に住む尻尾の生えた少年・孫悟空はある日、西の都からやって来た少女・ブルマと出会う。",
    },
    {
      button: "ピッコロ大魔王編",
      children:
        "天下一武道会終了後、ピラフ一味によって復活したピッコロ大魔王によって仲間たちが危機に陥る。",
    },
    {
      button: "サイヤ人編",
      children:
        "息子の孫悟飯を救うため、悟空は宿敵ピッコロと手を組んで強敵に挑む。",
    },
  ],
  [],
)

return <NativeAccordion.Root items={items} />
```

### Set a Specific Item to be Expanded by Default

To set a default expanded item, set `open` on `NativeAccordion.Item`.

```tsx preview
<NativeAccordion.Root>
  <NativeAccordion.Item button="孫悟空少年編">
    地球の人里離れた山奥に住む尻尾の生えた少年・孫悟空はある日、西の都からやって来た少女・ブルマと出会う。
  </NativeAccordion.Item>

  <NativeAccordion.Item button="ピッコロ大魔王編" open>
    天下一武道会終了後、ピラフ一味によって復活したピッコロ大魔王によって仲間たちが危機に陥る。
  </NativeAccordion.Item>

  <NativeAccordion.Item button="サイヤ人編">
    息子の孫悟飯を救うため、悟空は宿敵ピッコロと手を組んで強敵に挑む。
  </NativeAccordion.Item>
</NativeAccordion.Root>
```

### Disable Animation

To disable panel animation, set `animate` to `false`.

```tsx preview functional
const items = useMemo<NativeAccordion.Item[]>(
  () => [
    { button: "孫悟空少年編", children: "第一話" },
    { button: "ピッコロ大魔王編", children: "第二話" },
    { button: "サイヤ人編", children: "第三話" },
  ],
  [],
)

return <NativeAccordion.Root animate={false} items={items} />
```
