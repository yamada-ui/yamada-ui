## テーマ

`FileButton`は、[単一パーツのコンポーネント](/styled-system/theming/component-styles#単一パーツのコンポーネント)です。

:::note
`FileButton`は、[Button](/components/forms/button)のスタイルを継承しています。
:::

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const FileButton: ComponentStyle<"FileButton"> = mergeStyle(Button)()
```
