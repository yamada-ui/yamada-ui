## Theming

The `YearPicker` is a [multi part component](/styled-system/theming/component-styles#multi-part-components).

:::note
`YearPicker` inherits the style from [DatePicker](/components/forms/date-picker).
:::

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::

```ts
export const YearPicker: ComponentMultiStyle<"YearPicker"> =
  mergeMultiStyle(DatePicker)()
```
