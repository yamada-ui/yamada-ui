`Style props`は、コンポーネントに`props`を渡すだけでコンポーネントのスタイルを変更する方法です。また、多くの便利なショートハンドを提供しており、開発効率を向上させています。

```tsx
<Box w="full" p="md" bg="warning" color="white">
  This is Box
</Box>
```
