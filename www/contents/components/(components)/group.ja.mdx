## 使い方

:::code-group

```tsx [package]
import { Group } from "@yamada-ui/react"
```

```tsx [alias]
import { Group } from "@/components/ui"
```

```tsx [monorepo]
import { Group } from "@workspaces/ui"
```

:::

```tsx
<Group />
```

:::tip
子要素をグルーピングする必要がない場合は、[HStack](/docs/components/h-stack)や[VStack](/docs/components/v-stack)の使用を検討してください。
:::

:::tip
`Group`コンポーネントを使用する前に、[ButtonGroup](/docs/components/button#grouping)や[AvatarGroup](/docs/components/avatar#grouping)等の各コンポーネントのグルーピング機能が使用できるか検討してください。  
:::

### 方向を変更する

方向を変更する場合は、`orientation`に`"vertical"`または`"horizontal"`を設定します。デフォルトでは、`"horizontal"`が設定されています。

```tsx preview
<VStack alignItems="flex-start">
  <Group>
    <Button>Button</Button>
    <Button>Button</Button>
    <Button>Button</Button>
  </Group>

  <Group orientation="vertical">
    <Button>Button</Button>
    <Button>Button</Button>
    <Button>Button</Button>
  </Group>
</VStack>
```

### 子要素を引き延ばす

子要素を横幅いっぱいに引き延ばす場合は、`grow`を`true`に設定します。

```tsx preview
<Group grow w="full">
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
</Group>
```

### 子要素を連結する

子要素を連結する場合は、`attached`を`true`に設定します。

```tsx preview
<VStack alignItems="flex-start">
  <Group attached>
    <Input />
    <IconButton
      variant="surface"
      aria-label="Add"
      focusVisibleRing="inside"
      icon={<PlusIcon fontSize="xl" />}
    />
  </Group>

  <Group attached>
    <Tag colorScheme="gray" variant="solid">
      @yamada-ui/react
    </Tag>
    <Tag colorScheme="lime" variant="solid">
      10k+
    </Tag>
  </Group>

  <Group attached orientation="vertical">
    <Button variant="outline" focusVisibleRing="inside">
      Button
    </Button>
    <Button variant="outline" focusVisibleRing="inside">
      Button
    </Button>
    <Button variant="outline" focusVisibleRing="inside">
      Button
    </Button>
  </Group>
</VStack>
```
