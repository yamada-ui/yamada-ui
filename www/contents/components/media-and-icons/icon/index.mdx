## Import

```ts
import { Icon } from "@yamada-ui/react"
```

## Usage

Here is an example using [React Icons](https://react-icons.github.io/react-icons/).

```tsx live=false
import { FaRobot } from "react-icons/fa"
```

```tsx
<Icon as={FaRobot} />
```

### Change fontSize

```tsx
<Wrap gap="md">
  <For each={["xl", "2xl", "3xl", "4xl", "5xl", "6xl"]}>
    {(fontSize, index) => <Icon key={index} as={FaRobot} fontSize={fontSize} />}
  </For>
</Wrap>
```

```tsx
  />
</Icon>
```

### Use `Lucide`

Yamada UI provides a convenient library to use [Lucide](https://lucide.dev/).

```ts
import { GhostIcon } from "@yamada-ui/lucide"
```

```tsx
<GhostIcon />
```

:::note
If you want to learn more about `@yamada-ui/lucide`, please check [here](/components/media-and-icons/lucide).
:::

### Use `Font Awesome`

Yamada UI provides a convenient library to use [Font Awesome](https://fontawesome.com/).

```ts
import { FontAwesomeIcon } from "@yamada-ui/fontawesome"
import { faPoo } from "@fortawesome/free-solid-svg-icons"
```

```tsx
<FontAwesomeIcon icon={faPoo} />
```

:::note
If you want to learn more about `@yamada-ui/fontawesome`, please check [here](/components/media-and-icons/fontawesome).
:::
