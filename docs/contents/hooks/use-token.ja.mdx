## インポート

```ts
import { useToken } from "@yamada-ui/react"
```

## 使い方

```tsx functional=true
const bg = useToken("colors", "primary")

return (
  <Box bg={bg} p="md" rounded="md" color="white">
    Primary for color tokens is "{bg}"
  </Box>
)
```

:::note
テーマをもっと学びたい場合は、[こちら](/styled-system/theming)をご覧ください。
:::
