Yamada CLIを使用して、カラーやフォントサイズなどのカスタマイズしたテーマのトークンの型を生成できます。

## インストール

ターミナルで次のコマンドのいずれかを実行します。

<PackageManagers
  packageNameOrCommand={{
    pnpm: "pnpm add --save-dev @yamada-ui/cli",
    npm: "npm install --save-dev @yamada-ui/cli",
    yarn: "yarn add --dev @yamada-ui/cli",
    bun: "bun add --dev @yamada-ui/cli",
  }}
/>

## 使い方

### 型定義を更新する

CLIを使用することで、[Style Props](/styled-system/style-props)が参照するテーマのトークンの型補完を更新することができます。

`package.json`にスクリプトを追加し、スクリプト内でカスタマイズしたテーマの場所を指定します。

```json title=package.json
{
  ...
  "scripts" : {
    ...
    "theme" : "yamada-cli tokens path/to/theme.ts",
  } 、
  ...
}
```

:::note status=warning
`path/to/theme.ts`を置き換えることを忘れないでください。
:::

:::note status=warning
更新した型定義は、インストールされる度に上書きされるため、インストールごとに再度実行する必要があります。
:::

先ほど、追加したスクリプトを実行します。

<PackageManagers
  packageNameOrCommand={{
    pnpm: "pnpm theme",
    npm: "npm run theme",
    yarn: "yarn theme",
    bun: "bun run theme",
  }}
/>

### テーマを生成する

CLIを使用することで、デフォルトのテーマをローカルに生成することができます。

<PackageManagers
  packageNameOrCommand={{
    pnpm: "pnpm yamada-cli theme ./theme",
    npm: "npm run yamada-cli theme ./theme",
    yarn: "yarn yamada-cli theme ./theme",
    bun: "bun run yamada-cli theme ./theme",
  }}
/>
