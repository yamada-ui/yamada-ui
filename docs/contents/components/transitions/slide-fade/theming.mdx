## Theming

The `SlideFade` is a [single part component](/styled-system/theming/component-styles#single-part-components).

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::

```ts
export const SlideFade: ComponentStyle<"SlideFade"> = {
  baseStyle: {
    w: "100%",
  },
  defaultProps: {
    offsetX: 0,
    offsetY: 8,
    reverse: true,
  },
}
```
