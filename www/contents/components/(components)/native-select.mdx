## Usage

:::note
`NativeSelect` is a native UI component provided by the browser (user agent). It utilizes the browser's default functionality to display a dropdown list. If you want to extend the functionality and use a more styled component, please check [Select](/components/forms/select).
:::

:::code-group

```tsx [package]
import { NativeSelect } from "@yamada-ui/react"
```

```tsx [alias]
import { NativeSelect } from "@/components/ui"
```

```tsx [monorepo]
import { NativeSelect } from "@workspaces/ui"
```

:::

```tsx
<NativeSelect.Root>
  <NativeSelect.Group>
    <NativeSelect.Option />
  </NativeSelect.Group>
  <NativeSelect.Option />
</NativeSelect.Root>
```

### Use items

```tsx preview functional
const items = useMemo<NativeSelect.Item[]>(
  () => [
    { label: "日比野カフカ", value: "日比野カフカ" },
    { label: "市川レノ", value: "市川レノ" },
    { label: "亜白ミナ", value: "亜白ミナ" },
    { label: "四ノ宮キコル", value: "四ノ宮キコル" },
  ],
  [],
)

return <NativeSelect.Root items={items} placeholder="Select a character" />
```

### Exclude placeholder from options

By default, a placeholder is included in the options. To exclude the placeholder from options, set `includePlaceholder` to `false`.

```tsx preview functional
const items = useMemo<NativeSelect.Item[]>(
  () => [
    { label: "日比野カフカ", value: "日比野カフカ" },
    { label: "市川レノ", value: "市川レノ" },
    { label: "亜白ミナ", value: "亜白ミナ" },
    { label: "四ノ宮キコル", value: "四ノ宮キコル" },
  ],
  [],
)

return (
  <NativeSelect.Root
    includePlaceholder={false}
    items={items}
    placeholder="Select a character"
  />
)
```

## Accessibility

If you are not using `Field.Root`, set `aria-label` or `aria-labelledby` to `NativeSelect.Root`.

```tsx
const items = useMemo<NativeSelect.Item[]>(
  () => [
    { label: "日比野カフカ", value: "日比野カフカ" },
    { label: "市川レノ", value: "市川レノ" },
    { label: "亜白ミナ", value: "亜白ミナ" },
    { label: "四ノ宮キコル", value: "四ノ宮キコル" },
  ],
  [],
)

return (
  <NativeSelect.Root
    items={items}
    placeholder="Select a character"
    aria-label="キャラクター"
  />
)
```

```tsx
const items = useMemo<NativeSelect.Item[]>(

return (
```

### ARIA Roles and Attributes

| Component            | Roles and Attributes | Usage                                                                                                                                                                     |
| -------------------- | -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `NativeSelect.Field` | `aria-describedby`   | If `NativeSelect.Root` is within a `Field.Root` and `Field.Root` has an `errorMessage`, `helperMessage`, or a `Field.ErrorMessage`, `Field.HelperMessage`, sets its `id`. |
|                      | `aria-disabled`      | Set to `"true"` when `isDisabled` is set.                                                                                                                                 |
|                      | `aria-invalid`       | Set to `"true"` when `isInvalid` is set.                                                                                                                                  |
|                      | `aria-label`         | If `aria-label` is not set, the `placeholder` value is set.                                                                                                               |
|                      | `aria-required`      | Sets to `"true"` if `required` is set.                                                                                                                                    |
