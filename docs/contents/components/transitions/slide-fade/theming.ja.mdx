## テーマ

`SlideFade`は、[単一パーツのコンポーネント](/styled-system/theming/component-styles#単一パーツのコンポーネント)です。

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const SlideFade: ComponentStyle<"SlideFade"> = {
  baseStyle: {
    w: "100%",
  },
  defaultProps: {
    offsetX: 0,
    offsetY: 8,
    reverse: true,
  },
}
```
