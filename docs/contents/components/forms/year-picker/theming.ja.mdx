## テーマ

`YearPicker`は、[複数パーツのコンポーネント](/styled-system/theming/component-styles#複数パーツのコンポーネント)です。

:::note
`YearPicker`は、[DatePicker](/components/forms/date-picker)のスタイルを継承しています。
:::

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const YearPicker: ComponentMultiStyle<"YearPicker"> =
  mergeMultiStyle(DatePicker)()
```
