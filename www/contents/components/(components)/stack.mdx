```tsx preview
<Stack>
  {Array.from({ length: 4 }).map((_, index) => (
    <Box key={index} bg="bg.contrast" color="fg.contrast" p="md">
      Box
    </Box>
  ))}
</Stack>
```

## Usage

:::code-group

```tsx [package]
import { Stack } from "@yamada-ui/react"
```

```tsx [alias]
import { Stack } from "@/components/ui"
```

```tsx [monorepo]
import { Stack } from "@workspaces/ui"
```

:::

```tsx
<Stack />
```

:::tip
If you want a vertical stack, use [VStack](/docs/components/v-stack).
:::

:::tip
If you want a horizontal stack, use [HStack](/docs/components/h-stack).
:::

:::tip
To stack child elements on top of each other, use [ZStack](/docs/components/z-stack).
:::

### Adding Separators Between Child Elements

```tsx preview
<Stack separator={<Separator />}>
  {Array.from({ length: 4 }).map((_, index) => (
    <Box key={index} bg="bg.contrast" color="fg.contrast" p="md">
      Box
    </Box>
  ))}
</Stack>
```
