## テーマ

`PinInput`は、[複数パーツのコンポーネント](/styled-system/theming/component-styles#複数パーツのコンポーネント)です。

:::note
`PinInput`は、[Input](/components/forms/input)のスタイルを継承しています。
:::

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const PinInput: ComponentMultiStyle = mergeMultiStyle(Input, {
  baseStyle: {
    container: {
      gap: "sm",
    },
    field: {
      textAlign: "center",
    },
  },

  variants: {
    unstyled: {
      field: {
        h: "auto",
      },
    },
  },

  sizes: {
    xs: {
      field: {
        fontSize: "xs",
        boxSize: "6",
        rounded: "sm",
      },
    },
    sm: {
      field: {
        fontSize: "sm",
        boxSize: "8",
        rounded: "md",
      },
    },
    md: {
      field: {
        fontSize: "md",
        boxSize: "10",
        rounded: "md",
      },
    },
    lg: {
      field: {
        fontSize: "lg",
        boxSize: "12",
        rounded: "md",
      },
    },
  },
})({ omit: ["addon", "sizes"] })
```
