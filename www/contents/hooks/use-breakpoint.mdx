---
title: useBreakpoint
description: "`useBreakpoint` is a custom hook that returns the current breakpoint. This hook monitors changes in the window size and returns the appropriate value."
storybook: hooks-usebreakpoint--basic
source: hooks/use-breakpoint
---

```tsx preview functional client
const breakpoint = useBreakpoint()

return <Text>The current breakpoint is "{breakpoint}".</Text>
```

## Usage

:::code-group

```tsx [package]
import { useBreakpoint } from "@yamada-ui/react"
```

```tsx [alias]
import { useBreakpoint } from "@/components/ui"
```

```tsx [monorepo]
import { useBreakpoint } from "@workspaces/ui"
```

:::

```tsx
const breakpoint = useBreakpoint()
```

:::note
The return value refers to the [breakpoint](/docs/theming/breakpoints) of the theme.
:::
