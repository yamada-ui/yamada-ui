Interpolation is a feature for referencing [CSS custom properties (variables)](/styled-system/css-custom-properties) or tokens from the [theme](/styled-system/theming) within the style system.

## Usage

If you have created [CSS custom properties (variables)](/styled-system/css-custom-properties) using the `vars` from [Style Props](/styled-system/style-props), you can reference the set `name` with a `$`.

```tsx
<Box
  p="md"
  rounded="md"
  color="white"
  vars={[{ name: "bg", token: "colors", value: "primary" }]}
  bg="$bg"
>
  Box
</Box>
```

## Referencing Theme Tokens

In Yamada UI's [Style Props](/styled-system/style-props), only the corresponding property's tokens from the [theme](/styled-system/theming) are referenced. For example, `borderRadius` references the `radii` tokens, but not the `spaces` tokens. By using `$`, you can reference tokens other than the corresponding ones.

```tsx
<Box p="md" rounded="$spaces.3" bg="primary" color="white">
  Box
</Box>
```

Also, by using `$` within a string, you can reference tokens from the [theme](/styled-system/theming). This is effective for [shorthand properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Shorthand_properties) such as [border](https://developer.mozilla.org/en-US/docs/Web/CSS/border).

```tsx
<Box p="md" rounded="md" border="1px solid $colors.primary" color="primary">
  Box
</Box>
```
