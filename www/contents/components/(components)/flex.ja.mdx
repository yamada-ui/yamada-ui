```tsx preview
<Flex gap="md">
  {Array.from({ length: 4 }).map((_, index) => (
    <Box key={index} bg="bg.contrast" color="fg.contrast" p="md">
      Box
    </Box>
  ))}
</Flex>
```

## 使い方

:::code-group

```tsx [package]
import { Flex } from "@yamada-ui/react"
```

```tsx [alias]
import { Flex } from "@/components/ui"
```

```tsx [monorepo]
import { Flex } from "@workspaces/ui"
```

:::

```tsx
<Flex />
```

:::tip
子要素を縦方向に積み重ねたい場合は、[VStack](/docs/components/v-stack)を使用します。
:::

:::tip
子要素を横方向に積み重ねたい場合は、[HStack](/docs/components/h-stack)を使用します。
:::

:::tip
子要素を複数行に折り返す場合は、[Wrap](/docs/components/wrap)を使用します。
:::

### 方向

```tsx preview
<Flex direction="column" gap="md">
  {Array.from({ length: 4 }).map((_, index) => (
    <Box key={index} bg="bg.contrast" color="fg.contrast" p="md">
      Box
    </Box>
  ))}
</Flex>
```

### 整列

```tsx preview
<Flex align="center" gap="md">
  <For each={["sm", "md", "lg"]}>
    {(p, index) => (
      <Box key={index} bg="bg.contrast" color="fg.contrast" p={p}>
        Box
      </Box>
    )}
  </For>
</Flex>
```

### 揃え方

```tsx preview
<Flex justify="space-between" gap="md">
  {Array.from({ length: 4 }).map((_, index) => (
    <Box key={index} bg="bg.contrast" color="fg.contrast" p="md">
      Box
    </Box>
  ))}
</Flex>
```

### 折り返し

```tsx preview
<Flex wrap="wrap" gap="md" maxW="xs">
  {Array.from({ length: 4 }).map((_, index) => (
    <Box key={index} bg="bg.contrast" color="fg.contrast" p="md">
      Box
    </Box>
  ))}
</Flex>
```

### スペースを追加する

```tsx preview
<Flex w="full" gap="md">
  <Box bg="bg.contrast" color="fg.contrast" p="md">
    Box
  </Box>

  <Spacer />

  <Box bg="bg.contrast" color="fg.contrast" p="md">
    Box
  </Box>
</Flex>
```
