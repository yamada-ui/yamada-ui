## Theming

The `ScaleFade` is a [single part component](/styled-system/theming/component-styles#single-part-components).

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::

```ts
export const ScaleFade: ComponentStyle<"ScaleFade"> = {
  baseStyle: {
    w: "100%",
  },
  defaultProps: {
    scale: 0.95,
    reverse: true,
  },
}
```
