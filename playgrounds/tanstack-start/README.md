# TanStack Start + Yamada UI

This is a [TanStack Start](https://tanstack.com/start) SSR playground for [Yamada UI](https://yamada-ui.com).

## Getting Started

From the root of the monorepo:

```bash
pnpm install
cd playgrounds/tanstack-start
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Features

- Server-side rendering (SSR) with hydration
- File-based routing via TanStack Router
- Color mode support with cookie persistence
- Full Yamada UI component library
