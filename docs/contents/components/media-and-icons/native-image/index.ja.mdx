## インポート

```ts
import { NativeImage } from "@yamada-ui/react"
```

## 使い方

```tsx
<NativeImage
  src="https://dragon-ball-official.com/assets/img/intro/intro_1.png"
  width="512px"
  height="512px"
  alt="ドラゴンボール"
/>
```
