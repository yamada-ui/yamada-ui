## リセットスタイルをカスタマイズする

テーマの`styles.resetStyle`を変更するだけ、リセットスタイルはカスタマイズできます。

`styles.resetStyle`は、スタイルオブジェクトまたはスタイルオブジェクトを返す関数を定義できます。

```tsx live=false
import { UIProvider, extendTheme, UIStyle } from "@yamada-ui/react"

const resetStyle: UIStyle = {
  th: {
    textAlign: "center",
    fontWeight: "semibold",
  },
}

const customTheme = extendTheme({ styles: { resetStyle } })()

const App = () => {
  return (
    <UIProvider theme={customTheme}>
      <YourApplication />
    </UIProvider>
  )
}
```

関数の場合は、`theme`と`colorMode`を取得できます。

```tsx live=false
import { UIProvider, extendTheme, UIStyle } from "@yamada-ui/react"

const resetStyle: UIStyle = ({ theme, colorMode }) => ({
  th: {
    textAlign: "center",
    fontWeight: "semibold",
  },
})

const customTheme = extendTheme({ styles: { resetStyle } })()

const App = () => {
  return (
    <UIProvider theme={customTheme}>
      <YourApplication />
    </UIProvider>
  )
}
```

:::note
テーマをもっと学びたい場合は、[テーマをカスタマイズする](/styled-system/theming/customize-theme)や[コンポーネントのスタイル](/styled-system/theming/component-styles)をご覧ください。
:::

## リセットスタイルを追加する

インストールで[個別](/getting-started/installation/individual)や[最小限](/getting-started/installation/minimal)を選択して、`@yamada-ui/core`をインストールしている場合、リセットスタイルのコンポーネントは呼び出されていません。

リセットスタイルを使いたい場合は、次の3つのことを行う必要があります。

1. アプリケーションのルートに`ThemeProvider`を追加する。
2. `ResetStyle`を追加する。
3. リセットスタイルを定義し、`ThemeProvider`の`theme`に渡す。

```tsx live=false highlight={3,4,5,6,7,11,12,15}
import { ThemeProvider, ResetStyle } from "@yamada-ui/core"

const resetStyle: UIStyle = {
  "*": {
    boxSizing: "border-box",
  },
}

const App = () => {
  return (
    <ThemeProvider theme={{ styles: { resetStyle } }}>
      <ResetStyle />

      <YourApplication />
    </ThemeProvider>
  )
}
```

[デフォルトのテーマ](/styled-system/theming/default-theme)のリセットスタイルを使いたい場合は、`@yamada-ui/theme`をインストールした後、`defaultTheme`か`baseTheme`を`ThemeProvider`に渡します。

<PackageManagers packageNameOrCommand="@yamada-ui/theme" />

```tsx live=false highlight={7,8,11}
import { ThemeProvider, ResetStyle } from "@yamada-ui/core"
import { defaultTheme, baseTheme } from "@yamada-ui/theme"

const App = () => {
  return (
    // `defaultTheme` or `baseTheme`
    <ThemeProvider theme={baseTheme}>
      <ResetStyle />

      <YourApplication />
    </ThemeProvider>
  )
}
```

:::note
`defaultTheme`は、コンポーネントのスタイルなど、すべてのトークンが定義されています。
`baseTheme`は、グローバルスタイルやカラーなどのトークンは定義されていますが、コンポーネントのスタイルが定義されていません。
:::
