## Import

```ts
import {
  Alert,
  AlertIcon,
  AlertTitle,
  AlertDescription,
} from "@yamada-ui/react"
```

- `Alert`: A wrapper component for the alert.
- `AlertIcon`: A component that displays the icon for the alert's status.
- `AlertTitle`: A component that displays the title of the alert.
- `AlertDescription`: A component that displays the description of the alert.

## Usage

```tsx
<Alert>
  <AlertIcon />
  <AlertTitle>セル</AlertTitle>
  <AlertDescription>
    か…完全体に………完全体になれさえすれば………！！！
  </AlertDescription>
</Alert>
```

### Changing the Status

The color scheme and icon change according to the status. The default is set to `info`.

:::note
The color scheme and icons for statuses can be customized in the config's `alert.statuses`. If you want to customize the config, please check [here](/styled-system/configure/customize-config).
:::

```tsx
<VStack>
  <Alert status="info">
    <AlertIcon />
    <AlertTitle>セル</AlertTitle>
    <AlertDescription>
      か…完全体に………完全体になれさえすれば………！！！
```

### Customize components

You can change the icons and color schemes for each status by [Customize Config](/styled-system/configure/customize-config).

```tsx live=false highlight={3,4,5,6,7,8,9,13}
import { UIProvider, extendConfig } from "@yamada-ui/react"

const customConfig = extendConfig({
  alert: {
    statuses: {
      success: { icon: CustomIcon, colorScheme: "pink" },
    },
  },
})

const App = () => {
  return (
    <UIProvider config={customConfig}>
      <YourApplication />
    </UIProvider>
  )
}
```
