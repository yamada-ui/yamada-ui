## Import

```ts
import { Box } from "@yamada-ui/react"
```

## Usage

```tsx
<Box p="md" rounded="md" bg="primary" color="white">
  This is the Box
</Box>
```

### Using `as`

By passing `as`, you can render it as a different element or component.

```tsx
<Box as="button" p="md" rounded="md" bg="danger" color="white">
  This is the button
</Box>
```
