`Button`は、アクセシビリティに関して[WAI-ARIA - Button Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/button/)に従います。

## キーボード操作

| キー             | 説明                                           | 状態 |
| ---------------- | ---------------------------------------------- | ---- |
| `Space`, `Enter` | フォーカスしているボタンをアクティブにします。 | -    |

## ARIAロールと属性

| コンポーネント | ロールと属性    | 使い方                       |
| -------------- | --------------- | ---------------------------- |
| `Button`       | `role="button"` | ボタンであることを示します。 |
