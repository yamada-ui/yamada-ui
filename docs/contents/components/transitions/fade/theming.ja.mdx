## テーマ

`Fade`は、[単一パーツのコンポーネント](/styled-system/theming/component-styles#単一パーツのコンポーネント)です。

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const Fade: ComponentStyle<"Fade"> = {
  baseStyle: {
    w: "100%",
  },
}
```
