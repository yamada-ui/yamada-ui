## Theming

The `PasswordInput` is a [multi part component](/styled-system/theming/component-styles#multi-part-components).

:::note
`PasswordInput` inherits the style from [Input](/components/forms/input).
:::

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
