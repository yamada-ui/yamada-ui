`Tag`は、`onClose`が設定されると閉じるボタンを表示します。この閉じるボタンの`aria-label`は、デフォルトで`"Close tag"`を設定しています。もし、`aria-label`を任意の値に変更したい場合は、`closeButtonProps`に`aria-label`を設定します。

```tsx live=false
<Tag onClose={() => {}} closeButtonProps={{ "aria-label": "Close custom tag" }}>
  Tag
</Tag>
```

## ARIAロールと属性

| コンポーネント                   | ロールと属性             | 使い方                                                         |
| -------------------------------- | ------------------------ | -------------------------------------------------------------- |
| `TagCloseButton` <Tag>内部</Tag> | `aria-label="Close tag"` | `onClose`が設定されている場合は、`"Close tag"`が設定されます。 |
