## テーマ

`DonutChart`は、[複数パーツのコンポーネント](/styled-system/theming/component-styles#複数パーツのコンポーネント)です。

:::note
`DonutChart`は、[PieChart](/components/data-display/pie-chart)のスタイルを継承しています。
:::

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const DonutChart: ComponentMultiStyle<"DonutChart"> = mergeMultiStyle(
  PieChart,
  {
    baseStyle: {
      label: {},
    },
  },
)()
```
