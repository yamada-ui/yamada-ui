## テーマ

`CloseButton`は、[単一パーツのコンポーネント](/styled-system/theming/component-styles#単一パーツのコンポーネント)です。

:::note
`CloseButton`は、[Button](/components/forms/button)のスタイルを継承しています。
:::

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const CloseButton: ComponentStyle<"CloseButton"> = mergeStyle(Button, {
  baseStyle: {
    rounded: "md",
    _active: {
      bg: ["blackAlpha.200", "whiteAlpha.200"],
    },
    _focusVisible: {
      boxShadow: "outline",
    },
    _hover: {
      bg: ["blackAlpha.100", "whiteAlpha.100"],
    },
  },

  sizes: {
    sm: {
      boxSize: 6,
      fontSize: "2xs",
    },
    md: {
      boxSize: 8,
      fontSize: "xs",
    },
    lg: {
      boxSize: 10,
      fontSize: "md",
    },
  },

  defaultProps: {
    size: "md",
  },
})({ omit: ["variants", "sizes", "defaultProps"] })
```
