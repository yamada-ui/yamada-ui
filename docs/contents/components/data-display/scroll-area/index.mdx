## Import

```ts
import { ScrollArea } from "@yamada-ui/react"
```

## Usage


### Change the timing of the scrollbar display

To change the timing of the scrollbar display, set `type` to `always`, `hover`, `scroll`, etc.

- `always`: The scrollbar is always displayed.
- `hover`: The scrollbar is displayed only when the element is hovered over.
- `scroll`: The scrollbar is displayed only when the element is scrolled.
- `never`: The scrollbar is not displayed.

### Change the duration it takes for the scrollbar to disappear

To change the duration it takes for the scrollbar to disappear, set `scrollHideDelay` to a numerical value (milliseconds).

```tsx
<ScrollArea
  type="scroll"
  scrollHideDelay={4000}
  h="xs"
  innerProps={{ as: VStack, gap: "md" }}
>
  <Heading size="sm">孫悟空少年編</Heading>

  <Text>
```
