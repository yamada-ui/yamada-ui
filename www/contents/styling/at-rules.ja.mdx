## 概要

CSSの[アットルール](https://developer.mozilla.org/ja/docs/Web/CSS/At-rule)は、CSSの動作を既定するものです。

Yamada UIでは、特によく使われる`@media`, `@container`, `@supports`と`@keyframes`の4つをサポートしています。

## 使い方

### @media

[@media](https://developer.mozilla.org/ja/docs/Web/CSS/@media)は、メディアクエリを使用して、特定のメディアタイプ（スクリーン、プリントなど）や条件（画面幅、解像度など）に基づいてスタイルを適用するために使用されます。これにより、異なるデバイスやビューポートサイズに対応したレスポンシブなデザインを実現することができます。

例えば、`@media`のメディア種別の`print`を使い、印刷プレビューモードで表示されるスタイルを変更する場合は、`type`に`"print"`を設定します。

```tsx preview
<Center
  borderWidth="1px"
  boxSize="4xs"
  p="md"
  _media={[{ type: "print", css: { color: "success" } }]}
>
  <Text>Print me</Text>
</Center>
```

上記のコードは、次のようなCSSが生成されます。

```css
.Center {
  /* ... */

  @media print {
    color: var(--ui-colors-success);
  }
}
```

`print`などの一部のメディア種別では便利なショートハンドが用意されています。利用可能なプロパティは[アットルール](/docs/styling/style-props#アットルール)をご覧ください。

```tsx preview
<Center borderWidth="1px" boxSize="4xs" p="md" _print={{ color: "success" }}>
  <Text>Print me</Text>
</Center>
```

#### 複数のクエリを使う

複数のクエリを使う場合は、配列内に複数の値を設定します。

```tsx preview
<Box
  w="full"
  p="md"
  color="fg.contrast"
  bg="bg.contrast"
  _media={[
    { maxW: "1440px", css: { bg: "success" } },
    { minW: "481px", maxW: "1280px", css: { bg: "warning" } },
    { maxW: "480px", css: { bg: "danger" } },
  ]}
>
  This is Box
</Box>
```

上記のコードは、次のようなCSSが生成されます。

```css
.Box {
  /* ... */

  @media (max-width: 1440px) {
    background: var(--ui-colors-success);
  }

  @media (min-width: 481px) and (max-width: 1280px) {
    background: var(--ui-colors-warning);
  }

  @media (max-width: 480px) {
    background: var(--ui-colors-danger);
  }
}
```

#### 任意のクエリを使う

任意のクエリを使う場合は、`query`に文字列を設定します。

```tsx preview
<Box
  w="full"
  p="md"
  color="fg.contrast"
  bg="bg.contrast"
  _media={[
    {
      query: "(min-width: 977px) and (max-width: 1440px)",
      css: { bg: "success" },
    },
  ]}
>
  This is Box
</Box>
```

### @container

[@container](https://developer.mozilla.org/ja/docs/Web/CSS/@container)は、コンテナクエリを使用して、特定のコンテナのサイズや条件に基づいてスタイルを適用するために使用されます。これにより、コンテンツが配置されているコンテナのサイズに応じてスタイルを調整することができます。

```tsx preview
<Center
  containerType="inline-size"
  p="md"
  borderWidth="1px"
  resize="horizontal"
  overflow="auto"
  boxSize="4xs"
>
  <Text
    _container={[{ minW: "320px", maxW: "560px", css: { color: "success" } }]}
  >
    Resize me
  </Text>
</Center>
```

上記のコードは、次のようなCSSが生成されます。

```css
.Center {
  container-type: inline-size;
  /* ... */

  @container (min-width: 320px) and (max-width: 560px) {
    color: var(--ui-colors-success);
  }
}
```

#### コンテナ名を指定する

```tsx preview
<Center
  containerName="parent"
  containerType="size"
  p="md"
  borderWidth="1px"
  resize="both"
  overflow="auto"
  boxSize="2xs"
>
  <Center
    containerName="child"
    containerType="size"
    p="md"
    borderWidth="1px"
    resize="both"
    overflow="auto"
    boxSize="4xs"
  >
    <Text
      _container={[
        {
          name: "parent",
          orientation: "portrait",
          css: { color: "success" },
        },
        { name: "child", orientation: "landscape", css: { color: "danger" } },
      ]}
    >
      Resize me
    </Text>
  </Center>
</Center>
```

上記のコードは、次のようなCSSが生成されます。

```css
.Center1 {
  container-name: parent;
  container-type: size;
  /* ... */
}

.Center2 {
  container-name: child;
  container-type: size;
  /* ... */

  @container parent (orientation: portrait) {
    color: var(--ui-colors-success);
  }

  @container child (orientation: landscape) {
    color: var(--ui-colors-danger);
  }
}
```

### @supports

[@supports](https://developer.mozilla.org/ja/docs/Web/CSS/@supports)は、ブラウザーがCSS機能に対応しているかによって、CSS宣言を指定するために使用されます。

```tsx preview
<Box
  _supports={[
    {
      css: {
        color: "success",
      },
      query: "(display: flex)",
    },
  ]}
>
  <Text>Supported flex</Text>
</Box>
```

上記のコードは、次のようなCSSが生成されます。

```css
.Box {
  /* ... */

  @supports (display: flex) {
    color: var(--ui-colors-success);
  }
}
```

### @keyframes

[@keyframes](https://developer.mozilla.org/ja/docs/Web/CSS/@keyframes)は、アニメーションの中間状態を定義するために使用されます。アニメーションは、指定された時間内に一連のスタイルの変更を実行します。

アニメーションは、別セクションにドキュメントがあります。詳しくは、[アニメーション](/docs/styling/animation)をご覧ください。

### 他のアットルール

他のアットルールを使う場合は、`css`で定義します。

```tsx
<Box
  css={{
    "@<identifier> (RULE)": {
      // Define the style you want to customize.
    },
  }}
>
  This is Box
</Center>
```
