## 概要

Yamada UIのコンポーネントでは、[CSSカスタムプロパティ(変数)](https://developer.mozilla.org/ja/docs/Web/CSS/Using_CSS_custom_properties)を作成・参照することができます。

:::warning
Yamada UIのスタイルシステムを使用する上でカスタムプロパティが必要になるケースは、ほとんどありません。この機能は、他のライブラリのコンポーネントにYamada UIのテーマトークンなどを参照したい場合に有効です。
:::

## 使い方

[CSSカスタムプロパティ(変数)](https://developer.mozilla.org/ja/docs/Web/CSS/Using_CSS_custom_properties)を作成するには、`--`プレフィックスが付いたプロパティを`Props`または`css`プロパティに定義します。

```tsx
<Box
  css={{
    "--bg": "colors.bg.contrast",
  }}
  {...{
    "--fg": "colors.fg.contrast",
  }}
/>
```

上記のコードは、次のような`CSS`が生成されます。

```css
.Box {
  --bg: var(--ui-colors-bg-contrast);
  --fg: var(--ui-colors-fg-contrast);
}
```

カスタムプロパティを参照するには、CSSの[var](https://developer.mozilla.org/ja/docs/Web/CSS/var)を使用するか、スタイルシステムの`{}`([インターポレーション](/docs/styling/interpolation))を使用します。

```tsx preview
<Box
  p="md"
  rounded="md"
  css={{
    "--bg": "colors.bg.contrast",
  }}
  {...{
    "--fg": "colors.fg.contrast",
  }}
  bg="var(--bg)"
  color="{fg}"
>
  Box
</Box>
```

:::warning
設定したカスタムプロパティは、設定した要素の子要素にのみ適用されます。
:::
