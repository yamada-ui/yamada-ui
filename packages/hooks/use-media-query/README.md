# @yamada-ui/use-media-query

> This is an internal utility, not intended for public usage.

## Installation

```sh
$ pnpm add @yamada-ui/use-media-query
```

or

```sh
$ yarn add @yamada-ui/use-media-query
```

or

```sh
$ npm install @yamada-ui/use-media-query
```
