
### 垂直方向にあわせる

垂直方向に合わせる場合は、`block`プロパティを使用します。

```tsx preview
<Box p="xl" rounded="md" borderWidth="1" h="2xs">
  <Bleed block="xl">
    <Center maxH="2xs" overflow="hidden">
      <Image
        src="https://hokuto-no-ken.jp/wp/wp-content/themes/hokuto_cms_40th/images/main_vs_bg_new.jpg"
        alt="北斗の拳"
        objectFit="cover"
        objectPosition="center"
        w="100%"
      />
    </Center>
  </Bleed>
</Box>
```

### 方向を指定する

特定の方向に合わせる場合は、`inlineStart`や`blockEnd`などのプロパティを使用します。

```tsx preview
<VStack>
  <Box p="xl" rounded="md" borderWidth="1" h="2xs">
    <Bleed inlineStart="xl">
      <Center h="4xs" overflow="hidden">
        <Image
          src="https://hokuto-no-ken.jp/wp/wp-content/themes/hokuto_cms_40th/images/main_vs_bg_new.jpg"
          alt="北斗の拳"
          objectFit="cover"
          objectPosition="center"
          w="100%"
        />
      </Center>
    </Bleed>
  </Box>

  <Box p="xl" rounded="md" borderWidth="1" h="2xs">
    <Bleed inlineEnd="xl">
```
