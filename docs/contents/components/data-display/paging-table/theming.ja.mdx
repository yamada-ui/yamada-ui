## テーマ

`PagingTable`は、[複数パーツのコンポーネント](/styled-system/theming/component-style#複数パーツのコンポーネント)です。

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const PagingTable: ComponentMultiStyle = mergeMultiStyle(Table, {
  baseStyle: {
    pagingControl: {
      gridTemplateColumns: "1fr 5fr 1fr",
    },
  },

  sizes: {
    sm: {
      pagingControl: {
        gap: 2,
      },
    },
    md: {
      pagingControl: {
        gap: 4,
      },
    },
    lg: {
      pagingControl: {
        gap: 6,
      },
    },
    xl: {
      pagingControl: {
        gap: 8,
      },
    },
  },
})()
```
