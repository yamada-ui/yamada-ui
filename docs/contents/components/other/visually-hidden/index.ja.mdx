## インポート

```ts
import { VisuallyHidden } from "@yamada-ui/react"
```

## 使い方

視覚的に要素を隠すために使用されますが、スクリーンリーダーではアクセス可能にします。デフォルトでは、`span`要素をレンダリングします。

```tsx
<Button>
  <VisuallyHidden>Checkmark</VisuallyHidden>
  <Check />
</Button>
```
