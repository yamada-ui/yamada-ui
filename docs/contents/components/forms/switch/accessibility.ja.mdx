`Switch`は、アクセシビリティに関して[WAI-ARIA - Switch Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/switch/)に従います。

## キーボード操作

| キー             | 説明                                       | 状態 |
| ---------------- | ------------------------------------------ | ---- |
| `Tab`            | 無効ではないスイッチをフォーカスします。   | -    |
| `Space`, `Enter` | スイッチのオン・オフの状態を切り替えます。 | -    |

## ARIAロールと属性

| コンポーネント | ロールと属性    | 使い方                                                                      |
| -------------- | --------------- | --------------------------------------------------------------------------- |
| `Switch`       | `role="switch"` | スイッチであることを示します。                                              |
|                | `aria-checked`  | スイッチがオンの場合は`"true"`を設定し、オフの場合は`"false"`を設定します。 |
