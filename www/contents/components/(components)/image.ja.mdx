## 使い方

:::code-group

```tsx [package]
import { Image } from "@yamada-ui/react"
```

```tsx [alias]
import { Image } from "@/components/ui"
```

```tsx [monorepo]
import { Image } from "@workspaces/ui"
```

:::

```tsx
<Image />
```

:::tip
フレームワークが画像コンポーネントを提供している場合は、そのコンポーネントを使用します。各フレームワークの画像コンポーネントとの統合は、[こちら](/docs/get-started#pick-your-framework)から使用しているフレームワークのドキュメントをご覧ください。
:::

:::tip
異なる表示やデバイスのシナリオに応じて代替の画像を提供する場合は、[Picture](/docs/components/picture)を使用します。代替の画像が不要な場合は、`Image`を使用します。
:::

:::tip
ユーザーを表すプロフィール写真やイニシャルのアイコンを表示する場合は、[Avatar](/docs/components/avatar)を使用します。
:::

:::tip
`alt`には画像が読み込まれない場合に表示する代替テキストを設定します。装飾的な画像や限られた情報価値しかない画像など、意味を持たない画像の場合は、`alt=""`を設定します。
:::
