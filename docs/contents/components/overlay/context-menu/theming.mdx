## Theming

The `ContextMenu` is a [multi part component](/styled-system/theming/component-styles#multi-part-components).

:::note
`ContextMenu` inherits the style from [Menu](/components/overlay/menu).
:::

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::

```ts
export const ContextMenu: ComponentMultiStyle = mergeMultiStyle(Menu, {
  baseStyle: {
    trigger: {},
  },
})()
```
