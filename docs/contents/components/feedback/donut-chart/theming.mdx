## Theming

The `DonutChart` is a [multi part component](/styled-system/theming/component-styles#multi-part-components).

:::note
`DonutChart` inherits the style from [PieChart](/components/feedback/pie-chart).
:::

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::

```ts
export const DonutChart: ComponentMultiStyle = mergeMultiStyle(PieChart)()
```
