## テーマ

`Textarea`は、[単一パーツのコンポーネント](/styled-system/theming/component-styles#単一パーツのコンポーネント)です。

:::note
`Textarea`は、[Input](/components/forms/input)のスタイルを継承しています。
:::

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const Textarea: ComponentStyle<"Textarea"> = mergeStyle(
  pickStyle(Input, "field"),
  {
    baseStyle: {
      lineHeight: "short",
      py: "2",
      verticalAlign: "top",
    },

    sizes: {
      xs: {
        minH: "20",
        py: "2",
      },
      sm: {
        minH: "20",
        py: "2",
      },
      md: {
        minH: "20",
        py: "2",
      },
      lg: {
        minH: "20",
        py: "3",
      },
    },

    variants: {
      unstyled: {
        h: "auto",
        px: "0",
        py: "0",
      },
    },
  },
)()
```
