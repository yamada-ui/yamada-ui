## Import

```ts
import { VisuallyHidden } from "@yamada-ui/react"
```

## Usage

It is used to visually hide elements, but it is accessible in screen readers. By default, it renders a `span` element.

```tsx
<Button>
  <VisuallyHidden>Checkmark</VisuallyHidden>
  <Check />
</Button>
```
