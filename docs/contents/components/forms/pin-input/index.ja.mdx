## インポート

```ts
import { PinInput, PinInputField } from "@yamada-ui/react"
```

- `PinInput`: 子要素(入力フィールド)を制御するラッパーコンポーネントです。
- `PinInputField`: 入力フィールドを表示するコンポーネントです。

:::note
`PinInputField`は、省略することができます。
:::

## 使い方

```tsx
<PinInput />
```

### バリアントを変更する

```tsx
<VStack>
  <PinInput variant="outline" />
  <PinInput variant="filled" />
  <PinInput variant="flushed" />
  <PinInput variant="unstyled" />
</VStack>
```

### サイズを変更する

```tsx
<VStack>
  <PinInput size="xs" />
  <PinInput size="sm" />
  <PinInput size="md" />
  <PinInput size="lg" />
</VStack>
```

### デフォルトの値を設定する

デフォルトの値を設定する場合は、`defaultValue`に文字列または数値を設定します。

```tsx
<VStack>
  <PinInput defaultValue="1234" />
  <PinInput defaultValue="123" />
</VStack>
```

### フィールド数を変更する

フィールド数を変更する場合は、`items`に数値を設定します。

```tsx
<VStack>
  <PinInput items={3} />
  <PinInput items={4} />
  <PinInput items={5} />
  <PinInput items={6} />
</VStack>
```

### ボーダーのカラーを変更する

ボーダーカラーを変更するには、`focusBorderColor`や`errorBorderColor`プロパティを使用します。

```tsx
<VStack>
  <PinInput focusBorderColor="green.500" />
  <PinInput isInvalid errorBorderColor="orange.500" />
</VStack>
```

### プレースホルダーを変更する

プレースホルダーを変更する場合は、`placeholder`に文字列を設定します。

```tsx
<PinInput placeholder="💩" />
```

### タイプを変更する

デフォルトでは、数値のみ入力が可能です。英数字をサポートする場合は、`type`に`alphanumeric`を設定します。

```tsx
<PinInput type="alphanumeric" />
```

### 入力完了時の動作を設定する

入力完了時の動作を設定するには、`onComplete`プロパティを使用します。

```tsx functional=true
const { page } = useLoading()

return <PinInput onComplete={() => page.start({ duration: 5000 })} />
```

### ワンタイムパスワードとして使用する

ワンタイムパスワード(`autocomplete="one-time-code"`)として使用するには、`otp`を`true`に設定します。

```tsx
<PinInput otp />
```

### 入力された値をマスクする

入力された値をマスクする場合は、`mask`を`true`に設定します。

```tsx
<PinInput mask />
```

### カスタムフィールドを使用する

カスタムフィールドを使用する場合は、`PinInputField`を使用します。

```tsx
<PinInput>
  <PinInputField />
  <PinInputField />
  <PinInputField />
  <PinInputField />
</PinInput>
```

### フォーカス管理を無効にする

デフォルトでは、フィールドが入力されると、自動的に次のフィールドにフォーカスは移動します。また、`BackSpace`キーが入力された場合は、前のフィールドにフォーカスされます。この制御を無効にする場合は、`manageFocus`を`false`に設定します。

```tsx
<PinInput manageFocus={false} />
```

### 無効化する

無効化する場合は、`isDisabled`を`true`に設定します。

```tsx
<VStack>
  <PinInput isDisabled />

  <PinInput>
    <PinInputField isDisabled />
    <PinInputField isDisabled />
    <PinInputField isDisabled />
    <PinInputField isDisabled />
  </PinInput>

  <FormControl
    isDisabled
    label="Please one-time password"
    helperMessage="Just sent you a one-time password to your e-mail address."
    errorMessage="one-time password is required."
  >
    <PinInput />
  </FormControl>
</VStack>
```

### 読み取り専用にする

読み取り専用にする場合は、`isReadOnly`を`true`に設定します。

```tsx
<VStack>
  <PinInput isReadOnly />
  <PinInput>
    <PinInputField isReadOnly />
    <PinInputField isReadOnly />
    <PinInputField isReadOnly />
    <PinInputField isReadOnly />
  </PinInput>
  <FormControl
    isReadOnly
    label="Please one-time password"
    helperMessage="Just sent you a one-time password to your e-mail address."
    errorMessage="one-time password is required."
  >
    <PinInput />
  </FormControl>
</VStack>
```

### 無効な入力にする

無効な入力にする場合は、`isInvalid`を`true`に設定します。

```tsx
<VStack>
  <PinInput isInvalid />
  <PinInput>
    <PinInputField isInvalid />
    <PinInputField isInvalid />
    <PinInputField isInvalid />
    <PinInputField isInvalid />
  </PinInput>
  <FormControl
    isInvalid
    label="Please one-time password"
    helperMessage="Just sent you a one-time password to your e-mail address."
    errorMessage="one-time password is required."
  >
    <PinInput />
  </FormControl>
</VStack>
```

### 制御する

```tsx functional=true
const { page } = useLoading()
const [value, onChange] = useState("")

const onComplete = () => page.start({ duration: 5000 })

return <PinInput value={value} onChange={onChange} onComplete={onComplete} />
```

### `React Hook Form`を使う

```tsx functional=true
type Data = { pinInput: string }

const {
  control,
  handleSubmit,
  watch,
  formState: { errors },
} = useForm<Data>()

const onSubmit: SubmitHandler<Data> = (data) => console.log("submit:", data)

console.log("watch:", watch())

return (
  <VStack as="form" onSubmit={handleSubmit(onSubmit)}>
    <FormControl
      isInvalid={!!errors.pinInput}
      label="Token"
      errorMessage={errors.pinInput ? errors.pinInput.message : undefined}
    >
      <Controller
        name="pinInput"
        control={control}
        rules={{
          required: { value: true, message: "This is required." },
          minLength: { value: 4, message: "This is required." },
        }}
        render={({ field }) => <PinInput {...field} />}
      />
    </FormControl>

    <Button type="submit" alignSelf="flex-end">
      Submit
    </Button>
  </VStack>
)
```
