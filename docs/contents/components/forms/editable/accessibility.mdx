Setting `aria-label` on `TabList` will make it readable by screen readers.

```tsx live=false
<Editable defaultValue="オッス！オラ悟空！" aria-label="My Editable">
  <EditablePreview />
  <EditableInput />
</Editable>
```

## ARIA Roles and Attributes

| Component          | Role and Attributes | Usage                                                   |
| ------------------ | ------------------- | ------------------------------------------------------- |
| `EditableInput`    | `aria-invalid`      | Set to `"true"` if `isInvalid` is set on `"Editable"`.  |
|                    | `aria-disabled`     | Set to `"true"` if `isDisabled` is set on `"Editable"`. |
|                    | `aria-readonly`     | Set to `"true"` if `isReadOnly` is set on `"Editable"`. |
|                    | `aria-required`     | Set to `"true"` if `isRequired` is set on `"Editable"`. |
| `EditableTextarea` | `aria-invalid`      | Set to `"true"` if `isInvalid` is set on `"Editable"`.  |
|                    | `aria-disabled`     | Set to `"true"` if `isDisabled` is set on `"Editable"`. |
|                    | `aria-readonly`     | Set to `"true"` if `isReadOnly` is set on `"Editable"`. |
|                    | `aria-required`     | Set to `"true"` if `isRequired` is set on `"Editable"`. |
