## キーボード操作

| キー         | 説明                                 | 状態 |
| ------------ | ------------------------------------ | ---- |
| `Tab`        | 選択されたボタンをフォーカスします。 | -    |
| `ArrowLeft`  | 無効ではない前のボタンを選択します。 | -    |
| `ArrowRight` | 無効ではない次のボタンを選択します。 | -    |
