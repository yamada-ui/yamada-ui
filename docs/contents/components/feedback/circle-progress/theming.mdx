## Theming

The `CircleProgress` is a [single part component](/styled-system/theming/component-styles#single-part-components).

If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::

```ts
export const CircleProgress: ComponentStyle<"CircleProgress"> = {
  baseStyle: {
    display: "inline-block",
    position: "relative",
    verticalAlign: "middle",
  },
}
```
