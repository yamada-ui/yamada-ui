## Import

<PackageManagers packageNameOrCommand="@yamada-ui/charts" />

:::note status=warning
`@yamada-ui/charts` is not included in `@yamada-ui/react`, so it needs to be installed separately.
:::

```ts
import { DonutChart } from "@yamada-ui/charts"
```

## Usage

:::note
`DonutChart` internally uses [Recharts](https://recharts.org/en-US/api/PieChart).
:::

```tsx functional=true
const mewtwo: CellProps[] = useMemo(
  () => [
    {
      name: "HP",
      value: 106,
      color: "green.500",
    },
    {
      name: "こうげき",
      value: 110,
      color: "red.500",
    },
    {
      name: "ぼうぎょ",
      value: 90,
      color: "blue.500",
    },
    {
      name: "とくこう",
      value: 154,
      color: "purple.500",
    },
    {
      name: "とくぼう",
      value: 90,
      color: "orange.500",
    },
    {
      name: "すばやさ",
      value: 130,
      color: "cyan.500",
    },
  ],
  [],
)

return <DonutChart data={mewtwo} />
```

### Adjust inner radius, outer radius

Set a percentage or number to `innerRadius` to adjust inner radius or `outerRadius` to adjust outer radius.

:::note
If a percentage is set, it is calculated as a percentage of the chart height.
If a number is set, it is calculated as `px` of the radius.
:::

```tsx functional=true
const mewtwo: CellProps[] = useMemo(

return <DonutChart data={mewtwo} innerRadius="70%" outerRadius="80%" />
```

### Adjust start angle, end angle

Set an angle to `startAngle` to adjust start angle or `endAngle` to adjust end angle.

:::note
The direction is anticlockwise with 3 o'clock as 0°. To go clockwise, set a negative value.
The default values are `90` for `startAngle` and `-270` for `endAngle` to go clockwise from 12 o'clock.
:::

```tsx functional=true
const mewtwo: CellProps[] = useMemo(
  () => [
    {
      name: "HP",
      value: 106,
      color: "green.500",
    },
    {
      name: "こうげき",
      value: 110,
      color: "red.500",
    },
    {
      name: "ぼうぎょ",
      value: 90,
      color: "blue.500",
    },
    {
      name: "とくこう",
      value: 154,
      color: "purple.500",
    },
    {
      name: "とくぼう",
      value: 90,
      color: "orange.500",
    },
    {
      name: "すばやさ",
      value: 130,
      color: "cyan.500",
    },
  ],
  [],
)

return <DonutChart data={mewtwo} startAngle={90} endAngle={225} />
```

To format label values, use `labelFormatter`.

```tsx functional=true
      name: "とくこう",
      value: 154,
      color: "purple.500",
    },
    {
      name: "とくぼう",
      value: 90,
      color: "orange.500",
    },
    {
      name: "すばやさ",
      value: 130,
      color: "cyan.500",
    },
  ],
  [],
)

return (
  <VStack>
    <DonutChart data={mewtwo} valueFormatter={(value) => `${value}P`} />

      data={mewtwo}
      withLabels
      labelFormatter={(value) => `${value}P`}
    />
  </VStack>
)
```
