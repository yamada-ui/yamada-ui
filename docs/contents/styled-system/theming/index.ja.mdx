---
title: テーマ
description: Yamada UIのテーマは、カスタマイズ性があり、アプリケーションのデザインの一貫性を担保します。
order: 2
with_children: true
---

テーマは、コンポーネントのスタイルやカラー・スペース・フォントサイズ・その他多くのプロパティのトークンが定義されている変更可能なオブジェクトです。

テーマのトークンは、`Style props`などで使用されます。各`Style props`の参照先は、[こちら](/styled-system/style-props)から確認してください。

:::note
Yamada UIは、すぐに使える[デフォルトのテーマ](/styled-system/theming/default-theme)があります。
:::
