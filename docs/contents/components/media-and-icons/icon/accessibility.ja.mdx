## ARIAロールと属性

| コンポーネント | ロールと属性         | 使い方                                       |
| -------------- | -------------------- | -------------------------------------------- |
| `Icon`         | `aria-hidden="true"` | 要素をアクセシビリティツリーから除外します。 |
|                | `role="img"`         | 画像であることを示します。                   |
