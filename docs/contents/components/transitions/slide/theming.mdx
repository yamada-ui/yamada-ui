## Theming

The `Slide` is a [single part component](/styled-system/theming/component-styles#single-part-components).

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::

```ts
export const Slide: ComponentStyle<"Slide"> = {
  baseStyle: {
    position: "fixed",
    zIndex: "fallback(jeice, 110)",
  },
  defaultProps: {
    placement: "right",
  },
}
```
