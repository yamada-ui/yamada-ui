```tsx preview
<Blockquote.Root citeUrl="https://ja.wikipedia.org/wiki/ドラゴンボール">
  地球の人里離れた山奥に住む尻尾の生えた少年・孫悟空はある日、西の都からやって来た少女・ブルマと出会う。そこで、7つ集めると神龍（シェンロン）が現れ、どんな願いでも一つだけ叶えてくれるというドラゴンボールの存在を、さらに育ての親である孫悟飯の形見として大切に持っていた球がその1つ「四星球（スーシンチュウ）」であることを知り、ブルマと共に残りのドラゴンボールを探す旅に出る。
</Blockquote.Root>
```

## 使い方

:::code-group

```tsx [package]
import { Blockquote } from "@yamada-ui/react"
```

```tsx [alias]
import { Blockquote } from "@/components/ui"
```

```tsx [monorepo]
import { Blockquote } from "@workspaces/ui"
```

:::

```tsx
<Blockquote.Root>
  <Blockquote.Content />
  <Blockquote.Caption>
    <Blockquote.Cite />
  </Blockquote.Caption>
</Blockquote.Root>
```


### アイコンを表示する

アイコンを表示する場合は、`icon`に`ReactNode`を設定します。

```tsx preview
<Blockquote.Root
  variant="plain"
  cite="Wikipedia"
  citeUrl="https://ja.wikipedia.org/wiki/ドラゴンボール"
  icon={<Blockquote.Icon />}
>
  地球の人里離れた山奥に住む尻尾の生えた少年・孫悟空はある日、西の都からやって来た少女・ブルマと出会う。そこで、7つ集めると神龍（シェンロン）が現れ、どんな願いでも一つだけ叶えてくれるというドラゴンボールの存在を、さらに育ての親である孫悟飯の形見として大切に持っていた球がその1つ「四星球（スーシンチュウ）」であることを知り、ブルマと共に残りのドラゴンボールを探す旅に出る。
</Blockquote.Root>
```

### 配置を変更する

配置を変更する場合は、`justify`に`"start"`または`"center"`または`"end"`を設定します。デフォルトは、`"start"`です。

```tsx preview
<VStack>
  <For each={["start", "center", "end"]}>
    {(justify, index) => (
      <Blockquote.Root
        key={index}
        cite="Wikipedia"
        citeUrl="https://ja.wikipedia.org/wiki/ドラゴンボール"
        justify={justify}
      >
        地球の人里離れた山奥に住む尻尾の生えた少年・孫悟空はある日、西の都からやって来た少女・ブルマと出会う。そこで、7つ集めると神龍（シェンロン）が現れ、どんな願いでも一つだけ叶えてくれるというドラゴンボールの存在を、さらに育ての親である孫悟飯の形見として大切に持っていた球がその1つ「四星球（スーシンチュウ）」であることを知り、ブルマと共に残りのドラゴンボールを探す旅に出る。
      </Blockquote.Root>
    )}
  </For>
</VStack>
```
