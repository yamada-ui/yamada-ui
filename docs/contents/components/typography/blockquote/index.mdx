## Import

```ts
import {
  Blockquote,
  BlockquoteContent,
  BlockquoteCaption,
  BlockquoteCite,
} from "@yamada-ui/react"
```

## Usage

```tsx
<Blockquote citeUrl="https://ja.wikipedia.org/wiki/ドラゴンボール">
  地球の人里離れた山奥に住む尻尾の生えた少年・孫悟空はある日、西の都からやって来た少女・ブルマと出会う。そこで、7つ集めると神龍（シェンロン）が現れ、どんな願いでも一つだけ叶えてくれるというドラゴンボールの存在を、さらに育ての親である孫悟飯の形見として大切に持っていた球がその1つ「四星球（スーシンチュウ）」であることを知り、ブルマと共に残りのドラゴンボールを探す旅に出る。
</Blockquote>
```

### Change Variant

```tsx
<VStack>
  <For
    each={[
      {
        variant: "solid",
        colorScheme: "primary",
      },
      {
        variant: "subtle",
        colorScheme: "secondary",
      },
      {
        variant: "plain",
      },
    ]}
  >
    {({ variant, colorScheme }, index) => (
      <Blockquote
        key={index}
        variant={variant}
        colorScheme={colorScheme}
        citeUrl="https://ja.wikipedia.org/wiki/ドラゴンボール"
      >
        地球の人里離れた山奥に住む尻尾の生えた少年・孫悟空はある日、西の都からやって来た少女・ブルマと出会う。そこで、7つ集めると神龍（シェンロン）が現れ、どんな願いでも一つだけ叶えてくれるというドラゴンボールの存在を、さらに育ての親である孫悟飯の形見として大切に持っていた球がその1つ「四星球（スーシンチュウ）」であることを知り、ブルマと共に残りのドラゴンボールを探す旅に出る。
      </Blockquote>
    )}
  </For>
</VStack>
```

### Display Citation


### Display Icon

To display an icon, set a `ReactNode` to `icon`.

```tsx
<Blockquote
  variant="plain"
  cite="Wikipedia"
  citeUrl="https://ja.wikipedia.org/wiki/ドラゴンボール"
  icon={
    <QuoteIcon
      color={["blackAlpha.600", "whiteAlpha.700"]}
      fontSize="xl"
      left="0"
      position="absolute"
      top="0"
    />
  }
  px="8"
>
  地球の人里離れた山奥に住む尻尾の生えた少年・孫悟空はある日、西の都からやって来た少女・ブルマと出会う。そこで、7つ集めると神龍（シェンロン）が現れ、どんな願いでも一つだけ叶えてくれるというドラゴンボールの存在を、さらに育ての親である孫悟飯の形見として大切に持っていた球がその1つ「四星球（スーシンチュウ）」であることを知り、ブルマと共に残りのドラゴンボールを探す旅に出る。
</Blockquote>
```

### Change Alignment

To change the alignment, set `justify` to `"start"`, `"center"`, or `"end"`. The default is `"start"`.

```tsx
<VStack>
  <For each={["start", "center", "end"]}>
    {(justify, index) => (
      <Blockquote
        key={index}
        cite="Wikipedia"
        citeUrl="https://ja.wikipedia.org/wiki/ドラゴンボール"
        justify={justify}
      >
        地球の人里離れた山奥に住む尻尾の生えた少年・孫悟空はある日、西の都からやって来た少女・ブルマと出会う。そこで、7つ集めると神龍（シェンロン）が現れ、どんな願いでも一つだけ叶えてくれるというドラゴンボールの存在を、さらに育ての親である孫悟飯の形見として大切に持っていた球がその1つ「四星球（スーシンチュウ）」であることを知り、ブルマと共に残りのドラゴンボールを探す旅に出る。
      </Blockquote>
    )}
  </For>
</VStack>
```
