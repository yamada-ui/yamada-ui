## テーマ

`CircleProgress`は、[単一パーツのコンポーネント](/styled-system/theming/component-styles#単一パーツのコンポーネント)です。

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const CircleProgress: ComponentStyle<"CircleProgress"> = {
  baseStyle: {
    display: "inline-block",
    position: "relative",
    verticalAlign: "middle",
  },
}
```
