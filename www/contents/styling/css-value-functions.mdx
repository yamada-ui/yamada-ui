## Calculation & Comparison

Yamada UI provides functions that make CSS calculation and comparison functions more convenient.

### calc

You can use CSS's [calc](https://developer.mozilla.org/en-US/docs/Web/CSS/calc) to reference and calculate tokens from the [theme](/docs/theming).

```tsx preview
<Center
  w="calc(lg / 2)"
  bg="bg.contrast"
  color="fg.contrast"
  p="md"
  rounded="md"
>
  Calc
</Center>
```

Also, if the token name is a number, like the tokens in [theme](/docs/theming)'s [spaces](/docs/theming/tokens/spaces), you reference it using `{}` ([interpolation](/docs/styling/interpolation)).

```tsx preview
<Center
  w="calc(xs + {spaces.2})"
  bg="bg.contrast"
  color="fg.contrast"
  p="md"
  rounded="md"
>
  Use interpolated token
</Center>
```

### min

Use CSS's [min](https://developer.mozilla.org/en-US/docs/Web/CSS/min) to set the smallest value from the given arguments.

If there is only one argument, a second value of `"100%"` is set.

```tsx preview
<VStack>
  <Center
    w="min(100%, lg)"
    bg="bg.contrast"
    color="fg.contrast"
    p="md"
    rounded="md"
  >
    Min
  </Center>

  <Center w="min(md)" bg="bg.contrast" color="fg.contrast" p="md" rounded="md">
    Omitted Min
  </Center>
</VStack>
```

### max

Use CSS's [max](https://developer.mozilla.org/en-US/docs/Web/CSS/max) to set the largest value from the given arguments.

If there is only one argument, a second value of `"100%"` is set.

```tsx preview
<VStack>
  <Center
    w="max(100%, 2xs)"
    bg="bg.contrast"
    color="fg.contrast"
    p="md"
    rounded="md"
  >
    Max
  </Center>

  <Center w="max(xs)" bg="bg.contrast" color="fg.contrast" p="md" rounded="md">
    Omitted Max
  </Center>
</VStack>
```

### clamp

Use CSS's [clamp](https://developer.mozilla.org/en-US/docs/Web/CSS/clamp) to constrain a value between an upper and lower bound.

If there are two arguments, a recommended value of `"100%"` is set.

```tsx preview
<VStack>
  <Center
    w="clamp(md, 100%, xl)"
    bg="bg.contrast"
    color="fg.contrast"
    p="md"
    rounded="md"
  >
    Clamp
  </Center>

  <Center
    w="clamp(md, lg)"
    bg="bg.contrast"
    color="fg.contrast"
    p="md"
    rounded="md"
  >
    Omitted Clamp
  </Center>
</VStack>
```

## Color

Yamada UI provides functions that easily mix colors together, lighten, and darken colors.

### mix

Use CSS's [color-mix](https://developer.mozilla.org/en-US/docs/Web/CSS/color_value/color-mix) to mix colors together.

You can specify two or three arguments. The [method](https://developer.mozilla.org/en-US/docs/Web/CSS/color-interpolation-method) can be omitted, and the default is `in srgb`.

```tsx preview
<Center
  bg="mix(red.500, blue.500)"
  w="full"
  height="2xs"
  p="md"
  rounded="md"
  color="white"
>
  "in srgb" + "red.500" + "blue.500"
</Center>
```

You can change the ratio by specifying a percentage.

```tsx preview
<Center
  bg="mix(in lab, orange.500 80%, purple.500 20%)"
  w="full"
  height="2xs"
  p="md"
  rounded="md"
  color="white"
>
  "in lab" + "orange.500 80%" + "purple.500 20%"
</Center>
```

:::warning
Make sure the percentages add up to `100%`.
:::

### tint

Use [mix](#mix) to lighten a color by mixing it with `#FFFFFF`.

```tsx preview
<Center
  preview
  bg="tint(purple.500, 50%)"
  w="full"
  height="2xs"
  p="md"
  rounded="md"
  color="white"
>
  Tint color
</Center>
```

### shade

Use [mix](#mix) to darken a color by mixing it with `#000000`.

```tsx preview
<Center
  bg="shade(yellow.500, 50%)"
  w="full"
  height="2xs"
  p="md"
  rounded="md"
  color="white"
>
  Shade color
</Center>
```

### transparentize

Use [mix](#mix) to make a color transparent by mixing it with `transparent`.

```tsx preview
<Center
  bg="transparentize(red.500, 50%)"
  w="full"
  height="2xs"
  p="md"
  rounded="md"
  color="white"
>
  Transparentize color
</Center>
```

### tone

Use [mix](#mix) to create a color based on a specified color and tone.

```tsx preview
<VStack>
  {TONES.map((tone) => (
    <Center
      key={tone}
      bg={`tone(purple.500, ${tone})`}
      w="full"
      height="6xs"
      p="md"
      rounded="md"
      color="white"
    >
      <Text
        color={`tone(purple.500, ${tone})`}
        as="span"
        filter="invert(100%) grayscale(100%) contrast(100)"
      >
        Tone {tone}
      </Text>
    </Center>
  ))}
</VStack>
```

## Gradient

Yamada UI provides functions that easily create gradients.

To add a gradient, set functions and values to `bgGradient`.

- `linear(<direction>, <from>, <to>)`
- `radial(<from>, <to>)`

You can also use other CSS gradient functions like `repeating-linear` or `conic`.

Shortcuts are available for `<direction>`.

```ts
{
  'to-t': 'to top',
  'to-tr': 'to top right',
  'to-r': 'to right',
  'to-br': 'to bottom right',
  'to-b': 'to bottom',
  'to-bl': 'to bottom left',
  'to-l': 'to left',
  'to-tl': 'to top left',
}
```

```tsx preview
<Box
  w="full"
  h="2xs"
  rounded="md"
  bgGradient="linear(to-r, green.200, pink.500)"
/>
```

### Customizing Colors

You can use both [color tokens](/docs/theming/tokens/colors) and [CSS color values](https://developer.mozilla.org/en-US/Web/CSS/color).

```tsx preview
<Box
  w="full"
  h="2xs"
  rounded="md"
  bgGradient="linear(to-r, #59a9e1, #f37bdf)"
/>
```

### Gradient Tokens

Gradient tokens can be defined in the theme.

```ts
import { ThemeTokens } from "@yamada-ui/react"

export const gradients: ThemeTokens = {
  blue: "linear(to-r, #132293, #2375b4)",
}
```

```tsx preview
<Box w="full" h="2xs" rounded="md" bgGradient="blue" />
```

:::warning
The [default theme](/docs/theming/tokens/gradients) does not define gradient tokens. If you want to use gradient tokens, please check [customization](/docs/theming/customization) the tokens.
:::

### Text Gradient

To add a gradient to text, set `bgGradient` and `bgClip` to `text`.

```tsx preview
<Heading
  size="xl"
  bgGradient="linear(to-r, orange.400, red.500)"
  bgClip="text"
  truncated
>
  クリリンのことか……クリリンのことかーーーっ！！！！！
</Heading>
```

### Responsive Gradient

Pass an object to `bgGradient` to support responsive styles.

```tsx preview
<Box
  w="full"
  h="2xs"
  rounded="md"
  bgGradient={{
    base: "linear(to-tr, teal.300, yellow.400)",
    lg: "linear(to-t, blue.200, teal.500)",
    md: "linear(to-b, orange.100, purple.300)",
  }}
/>
```

:::note
If you want to learn more about responsive styles, please check [here](/docs/styling/responsive-design).
:::
