`Breadcrumb` follows the [WAI-ARIA - Breadcrumb Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/breadcrumb/) for accessibility.

When using `Breadcrumb`, set the `aria-label` for `Breadcrumb`.

```tsx live=false
<Breadcrumb aria-label="Breadcrumb">
  <BreadcrumbItem>
    <BreadcrumbLink href="/">サイヤ人編</BreadcrumbLink>
  </BreadcrumbItem>
  <BreadcrumbItem>
    <BreadcrumbLink href="/">ナメック星編</BreadcrumbLink>
  </BreadcrumbItem>
  <BreadcrumbItem>
    <BreadcrumbLink href="/">人造人間編</BreadcrumbLink>
  </BreadcrumbItem>
  <BreadcrumbItem isCurrentPage>
    <BreadcrumbLink href="/">魔人ブウ編</BreadcrumbLink>
  </BreadcrumbItem>
</Breadcrumb>
```

## ARIA Roles and Attributes

| Component        | Role and Attributes       | Usage                                                                        |
| ---------------- | ------------------------- | ---------------------------------------------------------------------------- |
| `Breadcrumb`     | `aria-label="Breadcrumb"` | If `aria-label` is not set, `"Breadcrumb"` will be set by default.           |
| `BreadcrumbLink` | `aria-current="page"`     | Set when `isCurrentPage` for `BreadcrumbItem` or `BreadcrumbLink` is `true`. |
