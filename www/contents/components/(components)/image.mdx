## Usage

:::code-group

```tsx [package]
import { Image } from "@yamada-ui/react"
```

```tsx [alias]
import { Image } from "@/components/ui"
```

```tsx [monorepo]
import { Image } from "@workspaces/ui"
```

:::

```tsx
<Image />
```

:::tip
If your framework provides an image component, use that component. For integration with each framework's image component, see the documentation for your framework [here](/docs/get-started#pick-your-framework).
:::

:::tip
Use [Picture](/docs/components/picture) to provide alternative images for different display or device scenarios. If you do not need alternative images, use `Image`.
:::

:::tip
To display a profile picture or an icon with initials representing a user, use [Avatar](/docs/components/avatar).
:::

:::tip
Set `alt` to alternative text that is displayed when the image fails to load. For images that do not convey meaning, such as decorative images or those with limited informational value, set `alt=""`.
:::
