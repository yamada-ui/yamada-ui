`Breadcrumb`は、アクセシビリティに関して[WAI-ARIA - Breadcrumb Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/breadcrumb/)に従います。

`Breadcrumb`を使用する場合は、`Breadcrumb`に`aria-label`を設定します。

```tsx live=false
<Breadcrumb aria-label="Breadcrumb">
  <BreadcrumbItem>
    <BreadcrumbLink href="/">サイヤ人編</BreadcrumbLink>
  </BreadcrumbItem>
  <BreadcrumbItem>
    <BreadcrumbLink href="/">ナメック星編</BreadcrumbLink>
  </BreadcrumbItem>
  <BreadcrumbItem>
    <BreadcrumbLink href="/">人造人間編</BreadcrumbLink>
  </BreadcrumbItem>
  <BreadcrumbItem isCurrentPage>
    <BreadcrumbLink href="/">魔人ブウ編</BreadcrumbLink>
  </BreadcrumbItem>
</Breadcrumb>
```

## ARIAロールと属性

| コンポーネント   | ロールと属性              | 使い方                                                                                |
| ---------------- | ------------------------- | ------------------------------------------------------------------------------------- |
| `Breadcrumb`     | `aria-label="Breadcrumb"` | `aria-label`が設定されていない場合は、`"Breadcrumb"`が設定されます。                  |
| `BreadcrumbLink` | `aria-current="page"`     | `BreadcrumbItem`または`BreadcrumbLink`の`isCurrentPage`が`true`の場合に設定されます。 |
