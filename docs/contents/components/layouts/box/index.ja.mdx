## インポート

```ts
import { Box } from "@yamada-ui/react"
```

## 使い方

```tsx
<Box p="md" rounded="md" bg="primary" color="white">
  This is the Box
</Box>
```

### `as`を使う

`as`を渡すことで、異なる要素やコンポーネントとしてレンダリングすることができます。

```tsx
<Box as="button" p="md" rounded="md" bg="danger" color="white">
  This is the button
</Box>
```
