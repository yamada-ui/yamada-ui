## Theming

The `FileButton` is a [single part component](/styled-system/theming/component-styles#single-part-components).

:::note
`FileButton` inherits the style from [Button](/components/forms/button).
:::

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::

```ts
export const FileButton: ComponentStyle<"FileButton"> = mergeStyle(Button)()
```
