## テーマ

`Table`は、[複数パーツのコンポーネント](/styled-system/theming/component-styles#複数パーツのコンポーネント)です。

:::note
`Table`は、[NativeTable](/components/data-display/native-table)のスタイルを継承しています。
:::

:::note
コンポーネントのスタイルを変更したい場合は、[こちら](/styled-system/theming/customize-theme#コンポーネントのスタイルを変更する)をご覧ください。
:::

```ts
export const Table: ComponentMultiStyle<"Table"> = mergeMultiStyle(
  NativeTable,
  {
    baseStyle: {
      sortIcon: {},
      td: {
        outline: "none",
        transitionDuration: "normal",
        transitionProperty: "common",
        _focusVisible: {
          boxShadow: "inline",
        },
      },
      th: {
        outline: "none",
        transitionDuration: "normal",
        transitionProperty: "common",
        _focusVisible: {
          boxShadow: "inline",
        },
      },
    },

    sizes: {
      sm: {
        sortIcon: {
          right: 2,
        },
      },
      md: {
        sortIcon: {
          right: 3,
        },
      },
      lg: {
        sortIcon: {
          right: 4,
        },
      },
      xl: {
        sortIcon: {
          right: 6,
        },
      },
    },
  },
)()
```
