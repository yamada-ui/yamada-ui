## Overview

Interpolation is a feature for referencing [CSS custom properties (variables)](/docs/styling/css-custom-properties) or tokens from the [theme](/docs/theming).

## Usage

When you create [CSS custom properties (variables)](/docs/styling/css-custom-properties), you can reference the configured `name` using `{name}`.

```tsx preview
<Box p="md" {...{ "--custom-bg-color": "#1ba14c" }} bg="{custom-bg-color}">
  Box
</Box>
```

### Referencing Theme Tokens

Yamada UI's [Style Props](/docs/styling/style-props) only reference tokens from the [theme](/docs/theming) for their corresponding properties. For example, `borderRadius` references tokens from `radii`, but does not reference tokens from `spaces`. By using `{}`, you can reference tokens other than the corresponding ones.

```tsx preview
<VStack>
  <Box p="md" rounded="md" border="1px solid">
    Box
  </Box>

  <Box p="md" rounded="{spaces.md}" border="1px solid">
    Box
  </Box>
</VStack>
```

Additionally, by using `{}` within strings, you can reference tokens from the [theme](/docs/theming). This is useful for [shorthand properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Shorthand_properties) such as [border](https://developer.mozilla.org/en-US/docs/Web/CSS/border).

```tsx preview
<Box p="md" border="{sizes.1} solid {colors.success}">
  Box
</Box>
```
