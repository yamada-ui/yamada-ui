## ARIA Roles and Attributes

| Component                            | Role and Attributes | Usage                                                                                                                               |
| ------------------------------------ | ------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `Pagination`                         | `role="navigation"` | Indicates that this is a navigation region between pages.                                                                           |
| `PaginationItem` <Tag>Internal</Tag> | `aria-label`        | Set each to `"Go to first page"`, `"Go to previous page"`, `"Go to next page"`, `"Go to last page"`, and `"Go to page <em>N</em>"`. |
