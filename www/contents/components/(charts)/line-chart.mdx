### Composition

```tsx preview functional
const data = useMemo<CartesianChartData[]>(() => createCartesianChartData(), [])

return (
  <LineChart.Root data={data}>
    <LineChart.Tooltip labelFormatter={(value) => dayjs(value).format("MMM")} />
    <LineChart.XAxis
      dataKey="date"
      tickFormatter={(value) => dayjs(value).format("MMM")}
    />
    <LineChart.Line dataKey="desktop" dot>
      <LineChart.LabelList />
    </LineChart.Line>
    <LineChart.Line dataKey="tablet" />

)
```

```tsx preview functional

  <VStack>
```

```tsx preview functional
      each={[
        "monotone",
        "bump",
        "linear",
        "natural",
        "step",
        "stepBefore",
        "stepAfter",
      ]}
    >
      {(type, index) => (
        <LineChart.Root
          key={index}
          lineProps={{ type }}

          series={series}
          tooltipProps={{
            labelFormatter: (value) => dayjs(value).format("MMM"),

            dataKey: "date",
```

### Add Unit

To add units, use `formatter`, `tickFormatter`, or set a string in `unit`.

```tsx preview functional

  () => LineChart.mergeSeries([{ dataKey: "desktop" }]),

  <LineChart.Root
    data={data}
    series={series}
    withYAxis
    chartProps={{ margin: { right: 16 } }}
    lineProps={{
      dot: true,
      label: {
        formatter: (value) => `${(Number(value) / 1000).toFixed(1)}k`,
      },
    }}
    tooltipProps={{
      formatter: (value) => `${(Number(value) / 1000).toFixed(1)}k`,
      labelFormatter: (value) => dayjs(value).format("MMM"),
    }}
    xAxisProps={{
      dataKey: "date",
      tickFormatter: (value) => dayjs(value).format("MMM"),
    }}
    yAxisProps={{
      domain: [0, 10000],
      tickFormatter: (value) => (value / 1000).toFixed(1),
      ticks: [0, 2500, 5000, 7500, 10000],
      unit: "k",
    }}
  />
)
```

```tsx preview functional

  <LineChart.Root
    data={data}
    series={series}
```

```tsx preview functional
    tooltipProps={{
      formatter: (value, name = "") => [
        Number(value).toLocaleString(),
        toTitleCase(name),
      ],
      labelFormatter: (value) => dayjs(value).format("MMM"),
    }}
    xAxisProps={{
      dataKey: "date",
      tickFormatter: (value) => dayjs(value).format("MMM"),
    }}
    yAxisProps={{
      tickFormatter: (value) => value.toLocaleString(),
    }}
  />
)
```

## Accessibility

### Keyboard Navigation

| Key          | Description                                                      | State                |
| ------------ | ---------------------------------------------------------------- | -------------------- |
| `ArrowRight` | Moves focus to the next data point and displays the tooltip.     | `withTooltip={true}` |
| `ArrowLeft`  | Moves focus to the previous data point and displays the tooltip. | `withTooltip={true}` |
| `Enter`      | Toggles the tooltip display for the focused data point.          | `withTooltip={true}` |
